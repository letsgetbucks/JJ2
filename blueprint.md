# Lotto Number Generator

## Overview

This project is a simple, visually appealing web application that generates and displays random lottery numbers. It is built using HTML, CSS, and JavaScript, following modern web development best practices.

## Design and Features

### Visual Design
*   **Layout:** A clean, centered layout that is mobile-responsive.
*   **Color Palette:** A vibrant and energetic color scheme with gradients and shadows to create depth and a "glow" effect on interactive elements.
*   **Typography:** Expressive fonts with clear hierarchy for headings and numbers.
*   **Iconography:** Icons will be used to enhance the user experience.
*   **Effects:** Subtle animations and transitions to make the interface more dynamic.

### Features
*   **Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Display:** Displays the generated numbers in visually distinct elements.
*   **Interactivity:** A button to trigger the number generation.

## Current Plan

*   **HTML:** Create the basic structure of the application, including a container for the numbers and a "Generate" button.
*   **CSS:** Style the elements with a modern and engaging design, including colors, fonts, shadows, and responsiveness.
*   **JavaScript:** Implement the logic for generating and displaying the lottery numbers when the button is clicked.
